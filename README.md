# ⚛ CORE REACTOR: Idle Clicker

A sci-fi idle clicker game — tap the reactor, build generators, prestige for Plasma Cores, and ascend through 7 tiers of boosts.

## 🎮 Play Now
**[Play CORE REACTOR](https://nicolastjacobs-png.github.io/core-reactor/)**


## ✨ Features
- **Click & Earn** — Tap the reactor core to generate energy
- **8 Generator Types** — Mining Drones to Dimension Rifts, orbiting the reactor
- **26 Tiered Boosts** — Unlocked across 7 prestige milestones
- **Prestige System** — Reset for Plasma Cores (+10% permanent multiplier each)
- **Trophy Bonuses** — 10 achievements, each granting permanent buffs
- **Offline Earnings** — Your reactors keep running while you're away
- **Cloud Save Ready** — Firebase scaffolding for cross-device sync
- **Full Settings** — Audio, visuals, dark/light theme, FPS cap, hold-to-click, hotkeys
- **Anti-Autoclicker** — CAPTCHA verification for suspicious click patterns
- **5 Languages** — English, Spanish, French, German, Japanese

## 🚀 Deploy on GitHub Pages
1. Create a new repo named `core-reactor`
2. Upload `index.html` and `README.md`
3. Settings → Pages → Source → main branch → Save
4. Live at `https://yourusername.github.io/core-reactor/`

## 📱 Works on desktop and mobile browsers — no install needed
